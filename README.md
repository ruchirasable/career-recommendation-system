# College Compass
AI-powered career guidance platform for JEE aspirants featuring IIT recommendations, cutoff prediction, aptitude assessment, and personalized college discovery.
## Application Preview
##  Features

-  Personalized College Recommendations
- IIT Cutoff Prediction
- Aptitude Assessment
-  User Authentication
- Data Visualization
